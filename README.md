# Community Campaign Manager（社区活动机器人）

Community Campaign Manager 是一个面向 Web3 社区的 Telegram 自动化活动管理工具，用于提升社区互动率并降低运营成本。

该机器人将原本需要人工执行的社区活动流程（如竞猜、抽奖和互动小游戏），转化为 **可自动执行、可管理、可复盘的社区运营能力**。

当前系统已支持：
- 币价竞猜活动
- 社区互动小游戏
- 活跃度抽奖
- 自动开奖与结果统计
- Web 后台管理
- 活动历史数据沉淀

其中 **币价竞猜模块** 是当前最成熟的核心功能。

---
# 项目背景

在币安社区中，社区天使需要持续组织互动活动，例如价格竞猜、抽奖或小游戏，以维持社区活跃度。

但传统模式通常依赖人工执行，例如：
- 手动发布活动
- 手动统计参与
- 手动整理开奖结果
- 手动公布获奖名单

这种方式存在一些问题：
- 人工成本较高
- 执行效率较低
- 统计容易出错
- 活动数据难以沉淀

因此，我使用 AI Agent 辅助开发快速构建了一套 Telegram 社区活动管理系统，帮助社区更加高效地组织互动活动。

---
# 系统架构

该项目不仅是一个 Telegram Bot，而是一套完整的社区活动管理系统。

系统主要由三个部分组成：

**Telegram Bot**
负责群内交互，例如发布活动、参与竞猜和公布结果。

**活动引擎**
负责活动规则管理、用户参与记录以及开奖结果计算。

**Web 管理后台**
提供可视化管理界面，用于创建活动、查看状态和查询历史记录。
数据库用于保存活动状态、用户参与数据和历史结果。

---

# 核心功能：币价竞猜
币价竞猜是当前系统最成熟的社区互动玩法。

管理员只需要设置：
- 竞猜币种
- 开奖时间
- 奖励规则

系统会自动发布竞猜活动。
用户在群内输入预测价格即可参与。

当开奖时间到达后，系统会自动：
1. 通过 Binance API 获取真实价格  
2. 计算用户预测误差  
3. 生成获奖名单  

最接近真实价格的用户将成为获胜者。

这种玩法具备：
- 参与门槛低
- 互动性强
- 适合周期性社区活动

# 自动化能力
系统实现了完整的活动自动化流程：
- 自动发布活动公告
- 自动记录用户参与
- 自动校验参与规则
- 自动获取市场价格
- 自动计算开奖结果
- 自动生成排行榜

如果活动设置为多轮模式，系统还可以自动开启下一轮活动。
所有活动数据都会保存，方便管理员查询与复盘。
