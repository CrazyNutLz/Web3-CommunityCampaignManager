# 社区活动机器人（Community Campaign Manager）

社区活动机器人是一个面向 Web3 社区的自动化活动管理工具，主要用于提升 Telegram 社区的互动率和运营效率。

机器人通过自动化系统，将原本需要人工执行的社区活动流程（如竞猜、抽奖和互动游戏）转化为可自动执行、可管理、可复盘的运营能力。

目前系统已实现：

- 币价竞猜活动
- 社区互动小游戏
- 活跃度抽奖
- 自动开奖与结果统计
- Web 后台活动管理
- 活动历史数据沉淀

其中 **币价竞猜模块** 是当前最成熟、最完整的功能模块。

# 项目背景

在币安社区中，币安天使们需要持续组织互动活动，例如价格竞猜、抽奖或小游戏，以维持社区活跃度。

但在传统模式下，这些活动往往需要天使们人工执行，例如：

- 手动发布活动
- 手动统计参与数据
- 手动整理开奖结果
- 手动公布获奖名单

这种方式存在明显问题：

- 人工成本较高
- 执行效率较低
- 统计过程容易出错
- 活动数据难以沉淀

所以，我通过 Agent 工具快速构建了一套 Telegram 社区活动管理系统，希望帮助社区更高效地组织互动活动并提升用户参与度。

# 系统方案

该项目不仅是一个 Telegram Bot，而是一套完整的社区活动管理系统。

系统主要由三个部分组成：

### Telegram Bot

负责群内交互，例如发布活动、参与竞猜和公布结果。

### 活动引擎

负责活动规则管理、用户参与记录以及开奖结果计算，是系统核心模块。

### Web 管理后台

提供可视化管理界面，用于创建活动、查看状态和查询历史记录。

数据库负责保存活动状态、用户参与数据以及开奖结果，使活动能够持续运行并保留历史记录。

# 核心功能：币价竞猜

币价竞猜是当前系统最成熟的社区活动玩法之一。

管理员只需设置：

- 竞猜币种
- 开奖时间
- 奖励规则

系统便会自动发布竞猜活动。

用户只需在群内输入预测价格即可参与。

当开奖时间到达后，系统会自动：

1. 通过币安API获取真实市场价格
2. 计算用户预测误差
3. 生成获奖名单

最接近真实价格的用户将成为获胜者。

这种玩法具有以下特点：

- 参与门槛低
- 社区互动性强
- 适合周期性活动运营

# 自动化能力

本项目实现了社区活动的自动化执行流程：

- 自动发布活动公告
- 自动记录用户参与
- 自动校验参与规则
- 自动获取市场价格
- 自动计算开奖结果
- 自动生成排行榜

如果活动设置为多轮模式，系统还可以在一轮结束后自动开启下一轮活动。

同时，所有活动数据都会被保存，方便管理员进行查询与复盘。

# 项目价值

从社区运营角度来看，该系统可以：

- 提升社区互动率
- 降低管理员运营成本
- 减少开奖统计错误
- 让社区活动流程更加规范
- 为未来积分体系和排行榜功能提供基础

简单来说，本项目将原本零散、人工驱动的社区活动流程，转化为可持续复用的自动化运营能力。
